<h1>osTicket - Prerequisites and Installation</h1>
This demonstration shows the installation of the open-source help desk ticketing system osTicket.<br />






<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Internet Information Services (IIS)

<h2>Operating Systems Used </h2>

- Windows 11</b> (PRO)

<h2>List of Prerequisites</h2>

- Turned on CGI
- Installed PHP Manager
- Installed IIS URL Rewrite Module 2
- Created PHP file in file explorer
- Installed Microsoft Visual C++
- Installed HeidiSQL
- Installed MySQL Server
- Registered new PHP version
- Installed and created osTicket account
